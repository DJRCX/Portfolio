# Portfolio – Mahtabul Al Nahian

Minimal portfolio built with **Next.js**, **Tailwind CSS**, and **Nord** (light theme). Uses **JetBrains Mono** for headings and names, and **Comfortaa** for body text.

## Getting Started

Follow these steps to set up the project fresh on any device.

### Prerequisites

- **Node.js**: Ensure you have Node.js installed (v18 or later recommended).
- **Git**: Ensure Git is installed to clone the repository.

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/portfolio.git
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd portfolio
    ```

3.  **Install dependencies:**

    ```bash
    npm install
    ```

4.  **Run the development server:**

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

```bash
npm run build
npm start
```

## Features

- **Nord light theme** – Snow Storm background, Polar Night text, Frost accents
- **Fonts** – JetBrains Mono (headings/names), Comfortaa (body/answers)
- **Sections** – Hero, About, Skills, Experience timeline, Projects, Contact
- **CV page** – [/cv](http://localhost:3000/cv), print-friendly
- **Schema.org** – JSON-LD for search visibility
- **Responsive** – Mobile menu and structured layout
