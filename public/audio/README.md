# Background music

Place your audio file here so the site can play it.

**Required file name:**  
`End_of_Wandering_Shorekeeper's_Piano_Theme.mp3`

If your file is still in the project root, move it here:

- From: `portfolio/End_of_Wandering_Shorekeeper's_Piano_Theme.mp3`
- To: `portfolio/public/audio/End_of_Wandering_Shorekeeper's_Piano_Theme.mp3`

The music starts automatically 4 seconds after the page loads.
